# 接口：getCourse  [返回](../README.md)
用例： [查询课程](../用例/查询课程.md)

- 功能：
    查询课程信息。
    
- 权限：
    管理员：查看课程信息，需先登录。    
    
- API请求地址： 
    接口基本地址/v1/api/getCourse/<search>

- 请求方式 ：
    GET
      
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |search|搜索内容|
  
- 返回实例：

        {         
            "status": true,
            "info": null,
            "data": [
                {"course_ID": "62435791",
                "course_name":"牛肉面制作工艺"
                "course_hour": "48"
                {
                ...其他用户
                }
            ]
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
  |course_ID|课程ID，对应表Course.course_ID的值|
  |course_name|课程名称|  
  |course_hour|课程课时|
