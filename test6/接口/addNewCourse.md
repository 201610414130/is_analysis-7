# 接口：addNewCourse  [返回](../README.md)
用例： [增加课程](../用例/增加课程.md)

- 功能：
    增加新用户。

- 权限：
    管理员：增加学生/老师用户。

- API请求地址：
    接口基本地址/v1/api/addNewCourse

- 请求方式 ：
    POST

- 请求实例：

        {
            "course_name": "牛肉面制作工艺",
            "course_hour":"48"
        }

- 请求参数说明:

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |course_name|课程名称|
  |course_hour|总课时|

- 返回实例：

        {         
            "status": true,
            "info": null
        }

- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
